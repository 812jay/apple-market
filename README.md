## 프로젝트 소개
사용자들간 중고물품을 거래하는 웹사이트 입니다.

## 사용 스킬
react, redux, node.js

## 환경설정
> server/config/dev.js 

- 개발자가 사용할 mongoDB URI를 설정합니다.

> npm i

- root 디렉토리, client 디렉토리 에서 package.json에 있는
패키지들을 설치합니다.
